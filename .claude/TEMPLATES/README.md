# Templates

Reusable templates for common project artifacts.

---

## Available Templates

| Template                                 | Purpose                      | Usage                                        |
| ---------------------------------------- | ---------------------------- | -------------------------------------------- |
| [plan.md](plan.md)                       | Task implementation plan     | Copy to `docs/plans/YYYY-MM-DD_task-name.md` |
| [pull-request.md](pull-request.md)       | Pull request description     | Paste into PR description                    |
| [bug-report.md](bug-report.md)           | Bug report issue             | Create bug issues                            |
| [feature-request.md](feature-request.md) | Feature request issue        | Create feature issues                        |
| [adr.md](adr.md)                         | Architecture Decision Record | Document major decisions                     |
| [incident-report.md](incident-report.md) | Post-incident report         | Document production incidents                |
| [release-notes.md](release-notes.md)     | Release announcement         | Public release communication                 |

---

## How to Use

1. Copy the template to appropriate location
2. Replace all `[placeholders]` with actual content
3. Remove sections that don't apply
4. Add project-specific sections if needed

---

## Template Conventions

- `[REQUIRED]` - Must be filled in
- `[OPTIONAL]` - Fill if applicable
- `<!-- comment -->` - Instructions, remove from final
- `YYYY-MM-DD` - Replace with actual date

---

_Templates are meant to be adapted. Modify for your project's needs._
